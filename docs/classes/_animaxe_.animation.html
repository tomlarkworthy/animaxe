Class Animation

An animation is pipeline that modifies the drawing context found in an animation Tick. Animations can be chained together to create a more complicated Animation. They are composeable,

e.g. animation1 = Ax.translate([50, 50]).fillStyle("red").fillRect([0,0], [20,20]) is one animation which has been formed from three subanimations.

Animations have a lifecycle, they can be finite or infinite in length. You can start temporally compose animations using anim1.then(anim2), which creates a new animation that plays animation 2 when animation 1 finishes.
